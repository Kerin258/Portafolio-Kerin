
# Portafolio - Kerin González

Proyecto de portafolio web desarrollado en React como entrega del parcial. El objetivo es presentar los trabajos realizados y servir como muestra para futuros empleadores.

Enunciado y material de apoyo: ver el enlace proporcionado por el curso con ejemplos y referencias.

**Características del proyecto**
- Interfaz responsiva y moderna.
- Sección de proyectos con tarjetas y descripciones.
- Sección "Sobre mí" y formulario de contacto.
- Animaciones y estilos con CSS/Vite/React.

**Instalación (local)**
1. Clona el repositorio:

	`git clone <URL-del-repositorio>`

2. Entra a la carpeta del proyecto:

	`cd portafolio-Kerin`

3. Instala dependencias:

	`npm install`

4. Inicia el servidor de desarrollo:

	`npm run dev`

Abre `http://localhost:5173` (u otra URL que muestre Vite) para ver el sitio en desarrollo.

**Comandos útiles**
- Ejecutar servidor de desarrollo: `npm run dev`
- Construir para producción: `npm run build`
- Probar la versión construida localmente: `npm run preview`


**Estructura principal**
- `index.html` — punto de entrada.
- `src/main.jsx` — arranque de React.
- `src/App.jsx` — componente raíz.
- `src/components/` — componentes (Hero, Projects, About, Contact, etc.).
- `src/portfolio.css`, `src/App.css`, `src/index.css` — estilos.
